#### 3、协作

接下来就是大家如何协作了。

- 首先，fork这个仓库
- 然后，clone到本地
- 在本地建个分支，然后在这个分支上进行自己的工作
- 觉得自己可以提交一些代码了，就先把分支push到自己的仓库
- 然后用这个分支对这个仓库发起一个pull request
- 等待合并
- 继续修改自己的代码，一步步优化，可以反复提交自己的代码



大致上就这些内容，希望大家合作愉快！当然这只是我的一个初步想法，具体的实施肯定有很多障碍，我们只需要一点点克服它们就好了！如果大家有什么想法也可以提出来。
